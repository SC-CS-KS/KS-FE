## 表格

```text
<table> 表示一个表格对象的开始
<tr> 表示表格一行的开始
<td> 表示表格中某行中的一个单元格的开始

<th> 标签定义表格的表头

```

<table>
    <tr>
        <th></th>
        <th>A</th>
        <th>B</th>
        <th>C</th>
    </tr>
    <tr>
        <th>1</th>
        <td>A1</td>
        <td>B1</td>
        <td>C1</td>
    </tr>
</table>
